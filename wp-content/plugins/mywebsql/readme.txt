MyWebSQL is the next generation database administration tool for the web. This plugin allows using MyWebSQL inside your WordPress admin panel to manage the WordPress database.

== Description ==

MyWebSQL is a rich gui client for database administration on the web. Intuitive and fast user interface, with keyboard shortcuts, desktop style layout, and lots of features to manage your database easily over the web.

This plugin allows using MyWebSQL inside your WordPress admini panel to manage your WordPress database, along with other databases on the server that the user has access to.

For full details and MyWebSQL feature list, please visit the project website at http://mywebsql.net

== Installation ==

1. Extract the contents of the plugin packages to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the WordPress `Tools` menu to access MyWebSQL
